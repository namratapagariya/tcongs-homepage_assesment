Getting Started
Clone the repo and install dependencies:

```bash git clone https://github.com/namratapagariya/tcongs-homepage_assesment.git cd tcongs-homepage_assesment npm install ```

Run the dev server:

```bash npm run dev ```

The site will be available at `http://localhost:5173\`.

Build for production:

```bash npm run build ```

Preview the production build locally:

```bash npm run preview ```
